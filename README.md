# TUGAS-BESAR-ALGORITMA-PEMROGRAMAN-2-KELOMPOK-15

Sistem Informasi Database Supplier Bahan Bangunan (BangunIn)

Deskripsi:

BangunIn adalah aplikasi untuk mengorganisir daftar mitra penyedia material konstruksi. Data utama yang digunakan adalah data profil supplier, data jenis material, dan data rating performa. Pengguna aplikasi adalah bagian pengadaan barang atau manajer logistik.

Spesifikasi:
1. Pengguna dapat menambahkan, mengubah, dan menghapus data supplier bahan bangunan.
2. Sistem dapat mencatat riwayat pelayanan dan detail kontak dari setiap mitra supplier.
3. Pengguna dapat mencari data supplier berdasarkan nama perusahaan atau lokasi menggunakan Sequential dan Binary Search. 
4. Pengguna dapat mengurutkan data mitra berdasarkan rating layanan tertinggi atau terendah menggunakan Selection dan Insertion Sort.
5. Sistem dapat menampilkan statistik jumlah supplier untuk setiap wilayah dan rata-rata skor kepuasan mitra.
